Who can log into the AMLHub Desktop?

  1. Visit the AMLHub desktop login page using the following URL https://amlhub.co.nz/login
  2. Login using the following credentials

    3. Username: Email Address

    Password: As you created it through the verification email, unless you have reset it since then (must be a minimum of 6 characters)

  3. You will be directed to the homepage, where it asks you to select an entity. Please click on the entity of which you would like to proceed with.
